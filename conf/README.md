# 基于React + React-Router 的演示项目
```
近一步优化思考： 基于node创建后台数据完成全栈数据。
```

## 1. start

```
$ git clone https://github.com/liangjingtao2016/router-shop.git
$ cd router-shop
$ npm install
```
## 2. dev
### 2.1 开发调试
```
//首先运行dll预打包
$ npm run start:pre
//打包完成，启动项目
$ npm start
```

查看效果` http://127.0.0.1:9090`

### 2.2 代码检查
```
$ npm run lint
```

### 2.3 测试用例
```
$ npm run test:watch
```

### 2.4 产出资源
```
$ npm run build
```
